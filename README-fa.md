# IPCleanScanner 🚀

[فارسی](README-fa.md) | [English](README.md)

IPCleanScanner یک ابزار **اسکن و تست IP تمیز برای Cloudflare / Fastly** در کنار کانفیگ‌های **v2rayN** است. کمک می‌کند IPهای تمیز را پیدا کنید و سپس **روی کانفیگ خودتان** تست کنید تا بهترین نتیجه را انتخاب کنید.

![اسکرین‌شات IPCleanScanner](screenshot.jpg)

## ✨ امکانات
- ⚡ اسکن سریع **IPهای تمیز Cloudflare / Fastly**
- 🧪 **تست IPهای پیدا شده** مستقیم روی **کانفیگ خودتان** (بر پایه Xray)
- 📄 بارگذاری **خروجی JSON** از v2rayN
- 🎯 مناسب برای جریان کاری **Cloudflare / Fastly**
- 🧩 روند ساده و عملی
- ✅ **اسکن رنج‌ها** بر اساس CIDR هر ارائه‌دهنده (Sequential / Random / Shuffle unique)
- 🎛️ **فیلتر حداکثر پینگ**، فیلتر Regex و محدودیت تعداد نتیجه
- 🧭 **تعویض خودکار رنج** بر اساس زمان، درصد اسکن‌شده یا اولین موفقیت
- 🧾 **خروجی نتایج** به صورت JSON یا CSV
- 🗂️ **ذخیره IPهای موفق** (همه یا به تفکیک ارائه‌دهنده)
- 🧪 **تست سرعت** با Ping / Download / Upload / Both
- 🌍 نمایش **Server IP + Country** بعد از آماده شدن پراکسی
- 🧠 **تشخیص خودکار ارائه‌دهنده** از CIDRهای `config/ip.json`
- 🧰 ابزارها: **CIDR → لیست IP**، مولد **IP → Config**
- 💾 **ذخیره/بازیابی تنظیمات برنامه** بین اجراها
- 📋 کپی سریع IPها از جدول نتایج

## ✅ این برنامه چه می‌کند؟
- IPهای تمیز Cloudflare / Fastly را اسکن و پیدا می‌کند
- امکان **تست همان IPها روی کانفیگ شما** برای ارزیابی واقعی را فراهم می‌کند

## 📦 خروجی گرفتن کانفیگ از v2rayN
1. **v2rayN** را باز کنید
2. روی کانفیگ **Cloudflare** یا **Fastly** راست‌کلیک کنید
3. گزینه **Export > Export selected for complete configuration** را بزنید
4. فایل **JSON** را ذخیره کنید
5. داخل IPCleanScanner بارگذاری کنید

## 🧭 روند پیشنهادی
1. خروجی JSON از v2rayN بگیرید
2. در IPCleanScanner لود کنید
3. اسکن IPهای تمیز Fastly را انجام دهید
4. IPهای پیدا شده را روی کانفیگ تست کنید
5. بهترین نتایج را انتخاب کنید

## 🧩 تب‌های اصلی (نمای کلی)
- **Ranges**: انتخاب ارائه‌دهنده‌ها، مدیریت CIDRها، بروزرسانی `config/ip.json`
- **Scan Range**: اسکن رنج‌ها با فیلترها و منطق تعویض خودکار
- **Scan IP**: تست لیست IP، خروجی گرفتن نتایج، ذخیره IPهای موفق
- **Speed Test**: تست مبتنی بر Xray (Ping / Download / Upload / Both)
- **Tools**: تبدیل CIDR، جابه‌جایی IPها بین تب‌ها، مولد IP → Config

## 🖥️ پیش‌نیازها
- Windows
- v2rayN (برای خروجی گرفتن کانفیگ)

## 📁 فایل‌ها
- `config/ip.json` شامل رنج‌های CIDR و آیکن‌های ارائه‌دهنده‌هاست
- `config/app_settings.json` برای ذخیره تنظیمات برنامه

## 📣 نکات
- حتما خروجی **complete configuration** از v2rayN بگیرید
- امکان بروزرسانی رنج‌ها از آدرس منبع ارائه‌دهنده وجود دارد
- برای موارد **اسکن IP تمیز Fastly** تست شده است
